# Step 10: React Hooks

In this step, you will learn about **React Hooks**.

Hooks allow function components to use React features such as:
- State
- Side effects
- Shared logic
- Performance optimization
- Global data sharing

Hooks are learned **one by one**, not all at once.

---

## How This Step Is Organized

Each hook:
- Has its **own folder**
- Has its **own README**
- Is taught **independently**

You should complete hooks **in order**.
