# Learn React Step by Step

This repository is designed to teach React from the very beginning.
It is written for people who have never learned React before.

You do not need prior React knowledge.
Each concept is explained slowly, clearly, and with simple examples.

---

## What is React?

React is a JavaScript library used to build user interfaces.
In simple terms, React helps you build interactive web pages using reusable pieces called components.

React is commonly used to build:
- Web applications
- Dashboards
- Admin panels
- Interactive user interfaces

---

## How to Use This Repository

This repository is divided into learning steps.
Each step has its own folder and its own README file.

You should follow the steps **in order**.
Do not skip steps, even if they feel simple.

If you are completely new, start from Step 1 below.
